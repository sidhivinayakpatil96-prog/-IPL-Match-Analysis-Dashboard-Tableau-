# -IPL-Match-Analysis-Dashboard-Tableau-
This project focuses on analyzing IPL match data using Tableau to derive meaningful insights about team performance, player statistics, and match trends.  The dashboard provides an interactive view of match-level and ball-by-ball data, enabling users to explore key metrics and patterns.

🎯 Objectives
Analyze team performance across matches
Identify top batsmen and bowlers
Understand the impact of toss decisions on match results
Track runs, wickets, and match trends

📊 Key Features
Interactive dashboard with filters (team, season, player)
KPIs: Total Runs, Wickets, Matches
Top performers (batsmen & bowlers)
Match outcome analysis
Toss vs Match result insights

🛠 Tools & Technologies
Tableau
SQL
Excel (Data Cleaning)

📈 Key Insights
Certain teams have higher win rates when winning the toss
Top players consistently contribute to match outcomes
Match trends highlight scoring patterns across games

🚀 Learnings
Improved data visualization and dashboard design
Hands-on experience with real-world dataset
Strengthened analytical thinking and storytelling
